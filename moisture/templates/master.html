SIS4MF

The SMART IRRIGATION SYSTEM FOR MIXED FARMING(SIS4MF) is a project aimed to designing a system that uses sensors to monitor soil moisture levels, and automatically adjusts the irrigation as needed. The system will be connected to a Web application that allows farmers to monitor and control the irrigation remotely.
